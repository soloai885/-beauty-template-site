# aigent.tw 現況分析與三行業版型設計規格（示範）

> 專案目標：以現有 aigent.tw（SoloAI LocalBiz Core）為基礎，針對「美學 / 美甲 / 美髮」三種常見店家型態，設計三套可延伸的網站版型（以首頁視覺稿示意），維持「線上預約」為核心轉換路徑。

## 1) 現況快速分析（aigent.tw）

### 1.1 資訊架構（可見頁）
- **入口首頁**：以「選擇你的美麗目的地」為主標，提供**類別卡片**導向不同店型。
- **店型示範頁（/beauty）**：
  - Hero 大圖 + 標語（強情境）
  - 主要 CTA：**會員預約 / 訪客預約**（與「預約查詢」並列）
  - 下方區塊（畫面顯示「精選服務」）→ 服務項目導購
- **共通元素**：右下客服浮動按鈕、右側快速工具（疑似切換/輔助）。

### 1.2 視覺語言（現況）
- **整體風格**：乾淨留白 + 大幅情境照片（美容空間、手部美甲）。
- **版面結構**：
  - 導覽列極簡（Logo / 入口按鈕）
  - Hero 以照片承載情緒與價值主張
  - CTA 使用高對比色塊按鈕
- **轉換路徑**：首頁 → 店型頁 → 會員/訪客預約（或預約查詢）

### 1.3 可優化的方向（做版型時會直接補強）
- **差異化敘事不足**：不同店型雖換圖，但「專業可信 / 服務特色 / 技師團隊 / 價格帶」的表達仍偏薄。
- **預約決策資訊**：建議在首頁版型中加入「熱門服務 / 價目區間 / 可預約時段 / 評價」等降低猶豫元素。
- **行業版型應該更像『專用模板』**：同一套系統、不同店型的「內容模組」排序與視覺母題（pattern）應更明顯。

## 2) 三套行業版型：共通設計原則

### 2.1 共通版型模組（所有行業都可用）
1. **Top Nav**：Logo（左）／「預約查詢」／主 CTA（立即預約）
2. **Hero**：情境照片/影片 + 一句價值主張 + 兩顆 CTA（會員/訪客 或 立即預約/看價目）
3. **熱門服務（3–6 張卡片）**：名稱、時間、價位區間、立即預約
4. **本店強項（3 點）**：專業/衛生/設備/材料（視行業替換）
5. **技師/團隊**：人像卡片（證照/年資/擅長）
6. **價格與方案**：套餐/加購（可收合）
7. **口碑**：Google/FB 評價摘要 + 客照（可選）
8. **位置與營業資訊**：地圖、交通、營業時間、聯絡
9. **Footer**：品牌資訊、條款、社群

### 2.2 UI 互動規格（共通）
- 首屏必見 CTA（不需捲動即可點到）。
- 服務卡：hover 顯示「可預約」或下一個空檔。
- 行動版：CTA 固定在底部（立即預約）。

## 3) 版型 A：美學 / 美體（Beauty & Wellness）

### 定位與語氣
- **情緒**：療癒、放鬆、儀式感。
- **主訴求**：專業護膚 + 舒壓體驗（降低壓力、提升狀態）。

### 視覺系統
- **色彩**：
  - 主色：玫瑰粉 / 珊瑚粉（溫度）
  - 輔色：奶油白、霧灰、柔和綠
- **字體建議**：
  - 標題：優雅襯線（或高質感黑體）
  - 內文：清晰黑體
- **影像**：自然光、植物、空間材質（木、布、石）。

### 模組排序（建議）
Hero → 熱門療程（臉/身）→ 皮膚/身體問題解法（3 格）→ 專業與儀器（可信度）→ 技師團隊 → 評價 → 位置

### 特色模組（行業專用）
- **「今天想改善什麼？」快速選單**：粉刺/乾燥/暗沉/緊緻/舒壓（點選後直接帶出推薦服務）。

## 4) 版型 B：美甲（Nail Art & Care）

### 定位與語氣
- **情緒**：精緻、潮流、可分享。
- **主訴求**：款式多、作品集強、預約快速。

### 視覺系統
- **色彩**：
  - 主色：深紫 / 霓虹紫（時尚）
  - 輔色：黑、冷灰、銀色高光
- **字體建議**：
  - 標題：俐落幾何黑體
  - 內文：中性黑體
- **影像**：手部特寫、作品拼貼、低景深、乾淨背景。

### 模組排序（建議）
Hero → **作品集 Masonry 拼貼**（先讓人心動）→ 熱門款式（可篩選：法式/鏡面/貓眼…）→ 價目（含卸甲/加固）→ 美甲師 → 評價 → 位置

### 特色模組（行業專用）
- **「按風格挑款」濾鏡**：
  - 風格標籤：極簡/甜酷/仙氣/派對/通勤
  - 點選即切換作品與推薦服務卡。

## 5) 版型 C：美髮（Hair Salon）— 本次補齊第三示範行業

> 由於目前公開頁未見第三個示範行業，本案以**美髮**作為最接近「預約型 LocalBiz」的第三模板示範（若你指定其他行業，如牙醫/餐飲/物理治療，我可同架構替換成對應模組與視覺）。

### 定位與語氣
- **情緒**：俐落、自信、專業。
- **主訴求**：剪染燙專長、設計師風格、可預約時段透明。

### 視覺系統
- **色彩**：
  - 主色：炭黑 / 深墨綠（專業質感）
  - 輔色：米白、香檳金（高端點綴）
- **字體建議**：
  - 標題：現代感黑體（較粗）
  - 內文：清爽黑體
- **影像**：髮型前後對比、設計師工作情境、乾淨棚拍。

### 模組排序（建議）
Hero → **設計師排班（本週空檔）** → 熱門服務（剪/染/燙/護）→ 風格提案（依髮長/髮質）→ 作品集 Before/After → 價目 → 評價 → 位置

### 特色模組（行業專用）
- **「依髮長/髮質推薦」**：短/中/長髮，細軟/受損/自然捲 → 對應推薦服務與設計師。
